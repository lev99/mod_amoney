# [AMoney](http://www.foto-s.ru/universal-module.html)

Module Donate - Amoney for Joomla! 2.5 and 3.x Version v1.1.0 with AJAX functionality

# It supports payment systems

* Webmoney
* RBK Money
* YandexMoney
* PayPal 
* Skrill | Moneybookers
* ROBOKASSA
* LiqPay 
* SMScoin 
* SavingsBank
